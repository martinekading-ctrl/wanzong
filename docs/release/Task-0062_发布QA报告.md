## 人工验收建议

1. 双击 `WanZong.exe`，确认主菜单主题、青玄宗图标与环境乐正常。
2. 开始新游戏，进入世界地图，检查地形、自然物、资源点和宗门图标。
3. 连续进入宗门并返回地图三次，确认镜头与数据稳定。

5. 手动保存，退出游戏，再从“继续游戏”恢复。
6. 在一台集成显卡 Windows 设备上复测窗口缩放、地图拖动和滚轮缩放。
