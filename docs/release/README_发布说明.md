# 《万宗》1.0.0 发布说明

## 运行环境

- Windows 10/11，64 位。
- 建议 1920×1080 分辨率；窗口缩放支持自适应。
- 使用 Godot 4.7 GL Compatibility 渲染，兼容集成显卡和较旧的 Windows 图形设备。

## 启动方法

保持以下文件位于同一目录，双击 `WanZong.exe`：

- `WanZong.exe`
- `WanZong.pck`

`WanZong.console.exe` 仅用于故障排查，普通玩家无需启动。

## 已完成玩法

- 主菜单、新游戏、继续游戏、完整存档/读档与自动存档。
- 6144×6144 烘焙世界地图、宗门、资源点、领地与地图对象交互。
- 玩家宗门、弟子名册、分工、修炼、突破、健康与装备。
- 宗门资源、建筑、升级、任务派遣、秘境探索与资源点占领。
- AI 宗门日/月模拟、发展、分裂、衰落、外交、联盟、附属、战争与议和。
- 自动战斗、战报、战争领地转移、背包、配方、炼丹炼器、市场贸易。
- 世界事件链、历史记录、长期目标、新手教程、正式 UI 主题与音频反馈。

## 基础操作

- 主菜单选择“开始游戏”进入新世界，或选择“继续游戏”读取最近进度。
- 世界地图可用鼠标拖动和滚轮缩放；点击宗门或资源点查看信息。
- 点击玩家宗门青玄宗，再点击“进入宗门”管理弟子、建筑、任务等系统。
- 宗门页顶部“推进一天”执行完整日结算。
- `Esc` 可取消世界地图当前选择。

## 音频替换

当前发布版内置轻量程序化环境乐与操作提示音。若后续提供正式音频，可放入 `res://assets/audio/bgm/` 并使用以下文件名，系统会优先加载：

- `main_menu.ogg`
- `world.ogg`
- `sect.ogg`
- `battle_theme.ogg`

也支持同名 WAV 或 MP3。

## 存档

存档写入 Godot 的用户数据目录，不会写入游戏安装目录。发布版提供三个手动槽、快速存档和自动存档。
